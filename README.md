## 抓取字段
+ user_id 对象ID
+ content 正文
+ comment_count 评论数
+ repost_count 转发数
+ favorite_count 点赞数
+ status_id 状态唯一ID
+ images 图片数组
+ video 视频地址
+ created_at 创建时间
+ is_repost 是否为转发微博 

## Setup and Run

# setup

```bash
$ pip3 install -r requirements.txt
```

# run

```bash
$ ./run.sh
Input weibo UID:
```
